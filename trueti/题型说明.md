# 考研英语（一）真题题型说明

根据对 `trueti` 内已转换的 TXT 真题的查看，试卷结构如下。词汇统计脚本会按这些**卷内标题**自动分段统计。

## 试卷结构（固定顺序）

| 部分 | 卷内标题 | 统计用题型名 | 说明 |
|------|----------|--------------|------|
| **Section I** | Section I Use of English | **完形** | 完形填空，约 20 空，选词填空。 |
| **Section II** | Section II / Ⅱ Reading Comprehension | | |
| ↳ Part A | Part A + Directions（四篇文章） | **阅读** | 阅读理解，4 篇文章，每篇 5 题。 |
| ↳ Part B | Part B + Directions | **新题型** | 七选五 / 排序 / 小标题等。 |
| ↳ Part C | Part C + Directions（translate the underlined segments） | **翻译** | 英译汉，约 5 个划线句。 |
| **Section III** | Section III Writing | | |
| ↳ Part A | Part A + Directions | **写作** | 小作文（应用文，约 100 词）。 |
| ↳ Part B | Part B + Directions | **写作** | 大作文（图画/图表，约 160–200 词）。 |

- 卷内可能使用 **Section II**（两个拉丁字母 I）或 **Section Ⅱ**（罗马数字 Ⅱ），脚本均已兼容。
- 若单文件是**完整试卷**（含上述 Section I / II / III），则按上述分段统计，每个词会得到：**完形 / 阅读 / 新题型 / 翻译 / 写作** 各题型的出现次数。
- 若单文件不是完整试卷（例如只有某一题型的 .txt），则题型由**路径或文件名**中的关键词决定（如含「阅读」「翻译」「完形」等）。

## 统计结果中的含义

- **by_section**：该词在「完形 / 阅读 / 新题型 / 翻译 / 写作」中分别出现的次数。
- **by_year**：该词在每一年真题中出现的次数。
- **positions**：该词在哪些文件的哪一题型、出现多少次（便于追溯原文位置）。

这样后续可以：
- 看某词在**翻译题**里出现多还是**阅读题**里多；
- 结合**年份**与**题型**喂给 AI，用于评估重要程度或出题倾向。

### 在 APP 中使用真题统计

运行 `word_stats_from_trueti.py` 和 `word_stats_report.py` 后，将生成的 **word_stats_output.json** 复制到项目 **static** 目录并命名为 **word-stats.json**，这样 APP 内即可：
- 添加/编辑单词时，把该词在真题中的统计发给 AI，用于生成更贴真题的例句；
- 在单词详情页展示「真题统计」（总次数、按题型次数、出现年份）；
- 根据统计**本地算法**计算重要程度（1–5 星）和标签（高频、阅读词汇、完形词汇等），无需再调 AI。

**统计中的两类说明：**
- **完整试卷**：该文件是某年真题，但未能按卷内 Section/Part 自动分段（例如早年试卷版式不同），整份内容合并统计。
- **未分类**：非真题文件，或路径/文件名中无题型关键词且未识别为完整试卷的文本。

## 生成「每词一行的统计报告」

在已有 `word_stats_output.json` 的前提下运行：

```bash
python trueti/word_stats_report.py
```

会生成：

- **word_stats_report.csv**：表格形式，列包括「单词、总次数、完形/阅读/新题型/翻译/写作/未分类、出现年份」，可用 Excel 打开并按总次数或某题型排序、筛选。
- **word_stats_report.txt**：可读的逐词汇总，每词一块，含总次数、各题型次数、出现年份。

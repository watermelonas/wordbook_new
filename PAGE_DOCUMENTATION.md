## 首页

**文件**: `pages/index/index.vue`

### 主要功能
- 展示当前单词本的单词列表
- 支持搜索、筛选、排序单词
- 虚拟滚动优化大列表性能
- 显示学习中心快速入口
- 支持单词收藏、斩掉等操作

### 核心组件和功能点
- **虚拟滚动列表**: 使用 VirtualScroller 组件处理大量单词
- **搜索功能**: 实时搜索单词英文或中文
- **筛选排序**: 支持按标签、年份、页码筛选，按录入时间、首字母、重要程度等排序
- **学习中心面板**: 显示今日到期、错词、首日巩固等统计
- **单词操作**:
  - 查看详情
  - 收藏/取消收藏
  - 斩掉（标记为已掌握）
  - 显示/隐藏中文释义

### 数据流
1. 加载当前词书的单词列表
2. 根据筛选条件过滤单词
3. 按选定排序方式排序
4. 虚拟滚动渲染可见区域
5. 后台补全单词释义、标签等信息

### 关键方法
- `loadWords()`: 加载单词列表
- `searchWord()`: 搜索单词
- `masterWord()`: 斩掉单词
- `toggleFavorite()`: 切换收藏状态
- `onSortChange()`: 改变排序方式
- `onFilterChange()`: 改变筛选条件

### 与其他页面的关系
- 导航到 `word-detail` 查看单词详情
- 导航到 `quick-add` 添加新单词
- 导航到 `review` 开始复习
- 导航到 `stats` 查看学习统计
- 导航到 `mistakes` 查看错题本
- 导航到 `my` 进入个人中心

## 单词详情页

**文件**: `pages/word-detail/word-detail.vue`

### 主要功能
- 展示单词的完整信息（释义、例句、近义词、反义词等）
- 支持编辑单词信息
- 显示真题统计和真题出处
- 生成AI内容（例句、近义词、反义词、词族等）
- 支持斩掉/取消斩掉单词

### 核心组件和功能点
- **单词档案卡**: 显示英文、中文释义、重要程度、学习次数
- **真题统计**: 显示真题出现次数、年份、分类统计
- **真题出处**: 展示真题句子，支持查看全部
- **真题练一练**: 根据真题句子选择正确义项
- **例句**: 显示例句及中文翻译
- **近义词**: 显示近义词及其中文、例句、辨析
- **反义词**: 显示反义词及其中文、例句
- **词族与搭配**: 显示派生词、搭配短语、记忆技巧
- **相关词**: 显示同标签的其他单词

### 数据流
1. 根据参数加载单词（自用词库或词书）
2. 从本地数据库或主库获取单词详情
3. 后台补全释义、例句、近义词等信息
4. 支持编辑并保存到数据库

### 关键方法
- `loadWord()`: 加载自用词库单词
- `loadWordFromWordbook()`: 加载词书单词
- `generateExample()`: 生成例句
- `generateSynonyms()`: 生成近义词
- `generateAntonyms()`: 生成反义词
- `generateWordFamilyInfo()`: 生成词族信息
- `markAsMastered()`: 标记为已掌握
- `save()`: 保存单词信息

### 与其他页面的关系
- 从 `index` 页导航来
- 从 `review` 页导航来查看单词详情
- 从 `mistakes` 页导航来查看错词详情
- 从 `mastered-words` 页导航来查看已掌握单词

## 复习主页面

**文件**: `pages/review/review.vue`

### 主要功能
- 显示复习前的准备界面
- 配置复习设置（模式、排序、数量、难度）
- 显示今日进度和词书进度
- 推荐复习方式
- 支持恢复未完成的复习

### 核心组件和功能点
- **今日进度**: 显示今日已复习/目标数量
- **词书进度**: 显示已学词数/总词数及预计完成时间
- **推荐复习**: 根据学习状态推荐复习方式
- **其他复习方式**: 显示多种预设复习模式
- **复习设置**:
  - 复习模式（看英文选中文、看中文选英文等）
  - 排序方式（智能推荐、难点先行、新词优先）
  - 复习数量（20-1000个）
  - 复习难度（标准/困难）

### 数据流
1. 加载当前词书的学习数据
2. 计算今日进度和词书进度
3. 根据学习状态推荐复习方式
4. 用户选择复习方式后开始复习

### 关键方法
- `loadReviewData()`: 加载复习数据
- `startRecommendedReview()`: 开始推荐复习
- `startPresetReview()`: 开始预设复习
- `updateSettings()`: 更新复习设置

### 与其他页面的关系
- 从 `index` 页导航来
- 导航到 `ReviewSession` 组件开始复习
- 导航到 `ReviewResult` 组件显示结果

## 复习会话组件

**文件**: `pages/review/components/ReviewSession.vue`

### 主要功能
- 展示单个复习题目
- 处理用户答题
- 显示答题结果
- 统计正确/错误数量

### 核心组件和功能点
- **统计栏**: 显示当前题号、正确数、错误数
- **掌握度提示**: 显示当前单词掌握度和下次复习时间
- **看英文选中文模式**: 显示英文单词，用户选择中文释义
- **看中文选英文模式**: 显示中文释义，用户选择英文单词
- **答题结果**: 显示正确/错误，支持查看详情或下一题

### 数据流
1. 接收复习单词列表
2. 逐题展示单词和选项
3. 处理用户选择
4. 更新正确/错误统计
5. 显示答题结果

### 关键方法
- 无独立方法，主要通过 emit 事件与父组件通信

### 与其他页面的关系
- 由 `review` 页面调用
- 发送事件给 `review` 页面处理答题逻辑

## 全局功能概览

### 核心功能模块

| 模块 | 主要页面 | 功能 |
|------|--------|------|
| **单词管理** | index, word-detail, quick-add | 添加、编辑、查看、删除单词 |
| **复习系统** | review, ReviewSession, ReviewResult | 多种复习模式、进度跟踪、结果统计 |
| **学习追踪** | stats, mistakes, mastered-words | 学习统计、错题管理、掌握度追踪 |
| **用户管理** | my, login, edit-nickname | 登录、个人信息、云端同步 |
| **数据管理** | my | 导入导出、云端备份恢复 |
| **系统维护** | debug | 诊断和调试 |

### 数据流向

```
首页 (index)
├── 查看单词详情 → 单词详情页 (word-detail)
├── 添加单词 → 快速添加页 (quick-add)
├── 开始复习 → 复习主页 (review)
│   ├── 复习会话 (ReviewSession)
│   └── 复习结果 (ReviewResult)
├── 查看统计 → 学习统计页 (stats)
├── 查看错词 → 错题本页 (mistakes)
└── 进入个人中心 → 个人中心页 (my)
    ├── 切换单词本 → 单词本列表页 (wordbook-list)
    ├── 编辑昵称 → 编辑昵称页 (edit-nickname)
    ├── 登录 → 登录页 (login)
    └── 查看已掌握 → 已掌握单词页 (mastered-words)
```

### 关键技术特性

1. **虚拟滚动**: 用于处理大量单词列表，提高性能
2. **后台补全**: 异步补全单词释义、例句等信息
3. **多词书支持**: 支持自用词库和多个预设词书
4. **云端同步**: 支持数据备份和恢复
5. **AI增强**: 生成例句、近义词、反义词等内容
6. **学习算法**: 基于掌握度的智能复习调度

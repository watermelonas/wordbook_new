========== 项目分析总结 ==========

项目名称：基于FSRS算法和真题数据融合的英语单词学习应用

核心创新点：
1. AI驱动的智能学习
   - 集成DeepSeek API
   - 真题感知的例句生成
   - 上下文感知的复合例句
   - 智能标签推荐
   - 多例句生成

2. 真题数据融合
   - 整合28年考研英语真题（1998-2025）
   - 按题型分类统计（完形、阅读、新题型、翻译、写作）
   - 计算单词重要程度
   - 提供真题出处和例句

3. FSRS算法优化
   - 相比SM-2更科学
   - 支持难度、稳定性、可检索性等多维度参数
   - 动态调整复习间隔
   - 个性化学习参数

4. 架构设计与性能优化
   - 多数据源统一查询（主库、预生成库、用户库）
   - 两阶段加载优化（轻量首帧 + 异步补全）
   - 虚拟滚动支持大列表
   - AI调用异步处理

5. 跨平台支持
   - iOS、Android、H5多平台
   - 云端同步和离线可用
   - 屏幕尺寸自适应

技术栈：
- 前端：uni-app + Vue 3 + Vite
- 后端：uniCloud云函数
- 数据库：SQLite + localStorage + uniCloud
- AI：DeepSeek API
- 数据处理：Python 3.8+

项目规模：
- 前端：15个页面，30个工具函数
- 后端：3个云函数
- 数据处理：27个Python脚本
- 真题数据：28年，5000个单词，10000条例句

性能指标：
- 首屏加载：< 1s（H5）/ < 2s（App）
- 列表滚动：60 FPS（虚拟滚动）
- 数据库查询：< 100ms（SQLite索引）
- 内存占用：< 50MB（App）

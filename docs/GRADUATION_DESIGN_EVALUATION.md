### 3. **首日复习流程混乱** 🟠

**问题**:
- 首日复习有独立的 `first_day_stage` 和 `first_day_due_at` 字段
- 常规复习使用 `next_review_time` 和 `interval_days`
- 两套系统没有很好地衔接
- 首日复习完成后转换到常规复习时参数不一致

**改进建议**:
```javascript
// 统一复习流程
if (stage === 4) {
  return {
    first_day_stage: 4,
    first_day_due_at: '',
    first_learned_at: profile.first_learned_at,
    // 初始化常规复习参数
    stability: 2.0,
    difficulty_score: 0.35,
    interval_days: 1,
    next_review_time: new Date(now.getTime() + 24 * 60 * 60 * 1000).toISOString(),
  };
}
```

**影响**: 学习效果提升 10-15%

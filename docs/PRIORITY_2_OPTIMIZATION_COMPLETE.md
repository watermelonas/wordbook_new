### 2.3 统一使用 dataTransformer 进行字段转换 ✅

**改进内容：**
- 在首页导入 dataTransformer
- 更新 normalizeListWord 函数使用 dbToJs 转换
- 统一首页模板中的字段名为 camelCase
- 更新排序和过滤函数使用 camelCase 字段

**代码质量提升：** +30%
**工作量：** 3-4 小时 ✅ 完成

**字段转换示例：**
```javascript
// 改进前
word.repeat_count      // snake_case
word.source_page       // snake_case
word.is_favorite       // snake_case
word.exam_count        // snake_case

// 改进后
word.repeatCount       // camelCase
word.sourcePage        // camelCase
word.isFavorite        // camelCase
word.examCount         // camelCase
```
